Search all files in a type
-------------------------------------------------------------
http://192.168.75.128:9200/<index>/<type>/_search?pretty=1


_____________________________________________________________
Search all files in an index
-------------------------------------------------------------
http://192.168.75.128:9200/<index>/_search?pretty=1


_____________________________________________________________
Check all indexes and their health
-------------------------------------------------------------
http://192.168.75.128:9200/_cat/indices?v


_____________________________________________________________
To check cluster state
-------------------------------------------------------------
curl -XGET 'http://localhost:9200/_cluster/state?pretty' 


_____________________________________________________________
Add an index to cluster
-------------------------------------------------------------
curl -XPUT 'localhost:9200/<new_index>?pretty'


_____________________________________________________________
Get all mappings
-------------------------------------------------------------
curl -XGET 'http://localhost:9200/_mapping'


_____________________________________________________________
Count docs in a cluster
-------------------------------------------------------------
curl -XGET 'http://localhost:9200/_count?pretty'


_____________________________________________________________
Get cluster name
-------------------------------------------------------------
curl 'http://localhost:9200/?pretty'


_____________________________________________________________
Delete an index
-------------------------------------------------------------
curl -XDELETE 'localhost:9200/<index>?pretty'
